# Auto Program Executor
작업에 필요한 프로그램과 웹사이트를 명령어 하나로 모두 실행시키는 프로그램

>## 명령어
>  > - help
>  >  > 명령어 목록을 나타냅니다.
>
>  > - exit
>  >  > 프로그램을 종료합니다.
>
>  > - clear
>  >  > 창의 모든 글자를 지웁니다.
>
>  > - ls
>  >  > 실행기의 목록을 나타냅니다.
>
>  > - mkexe (실행기 이름)
>  >  > 실행기를 생성합니다.
>
>  > - rmexe (실행기 이름)
>  >  > 실행기를 삭제합니다.
>
>  > - modexe (실행기 이름) (-a or -r) (내용)
>  >  > 실행기의 내용을 추가/제거합니다.
>  >  > 내용은 W-(웹사이트 링크) 혹은 P-(.exe 파일 경로) 입니다.
>  >  > 웹사이트 링크 혹은 파일 경로에 공백이 포함된다면 공백을 #으로 작성해야 합니다.
>
>  > - exels (실행기 이름)
>  >  > 실행기의 내용 목록을 나타냅니다.
>
>  > - run (실행기 이름)
>  >  > 실행기를 실행합니다.
